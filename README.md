# Shoppy

Smart Shopping Companion

## App-Idee

Shoppy ist eine mobile Expo-App, die Einkaufsvorbereitung einfacher macht. Nutzerinnen und Nutzer sollen Einkaufslisten schnell erfassen, organisieren und im Laden effizient abarbeiten koennen.

## Zielgruppe

Die App richtet sich an Studierende, Familien und Berufstaetige, die regelmaessig einkaufen und ihre Einkaufsplanung uebersichtlich digitalisieren moechten.

## Wichtigste Funktionen

- Einkaufslisten erstellen und bearbeiten
- Artikel nach Status markieren
- Uebersichtlicher Startbildschirm mit aktuellem Listenstatus
- Spaeter geplant: Kategorien, Favoriten, Budget-Hinweise und geteilte Listen

## Runbook

Projekt installieren:

```bash
npm install
```

Projekt starten:

```bash
npm start
```

Alternativ direkt mit Expo:

```bash
npx expo start
```

Danach kann die App mit Expo Go auf einem Smartphone oder in einem Emulator gestartet werden.
